Detailed Description

A destination class.

Parameters
destinationis type string keeping the location.
fromis type Date keeping the start date.
tois type Date keeping the end date.
gradeis type unsigned.
commentis type string.
photosis a vector of strings.

This class is used to initialize trips. They are part of every user's database. Every trip consists of location, start date, end date, user's grade, user's comment and photos. Class date is used to maintain the dates in class Destination.

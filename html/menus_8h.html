void registration ( )

This function is used to create a profile. Firstly, an object of type User is created with default values(empty strings for username, password and email). Then asking the user to input these values and through the setters of class User the inserted values are assigned to the object. There is a validation to prevent registering two profiles with the same username or email. Moreover two files are being opened and through the getters of class User all the data of the object is saved into the files.
